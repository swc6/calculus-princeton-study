## 导数的定义

### 选择题
1. 函数$f(x)$在$x = a$处可导的充要条件是：
   A. $f(x)$在$x = a$处连续
   B. $f(x)$在$x = a$处左导数和右导数都存在
   C. $f(x)$在$x = a$处左导数和右导数都存在且相等
   D. $f(x)$在$x = a$处有定义

2. 若$f'(x_0) = 2$，则$\lim_{h \to 0} \frac{f(x_0 + h) - f(x_0 - h)}{2h} = $
   A. 1
   B. 2
   C. 4
   D. 0

3. 函数$y = |x - 1|$在$x = 1$处：
   A. 连续且可导
   B. 连续但不可导
   C. 不连续但可导
   D. 不连续也不可导

### 填空题
1. 设$f(x) = x^3$，则$f'(2) = $ __________

2. 曲线$y = x^2$在点$(1, 1)$处的切线方程为 __________

3. 若$f(x) = \sin x$，则$f'\left(\frac{\pi}{4}\right) = $ __________

### 解答题
1. 用导数的定义求函数$f(x) = x^2 - 2x + 1$在$x = 1$处的导数。

2. 用导数的定义求函数$f(x) = \sqrt{x}$的导数。

3. 讨论函数$f(x) = \begin{cases} x^2 \sin \frac{1}{x} & x \neq 0 \\ 0 & x = 0 \end{cases}$在$x = 0$处的连续性和可导性。

## 综合应用题

1. 求曲线$y = x^3 - 3x^2 + 2$在点$(1, 0)$处的切线方程和法线方程。

2. 已知物体的运动方程为$s = t^3 - 3t^2 + 2t$，求物体在$t = 2$时的速度和加速度。

3. 设生产$x$单位产品的成本函数为$C(x) = 100 + 0.01x^2$，求边际成本函数和当$x = 100$时的边际成本。

4. 求函数$f(x) = x^2 e^{-x}$的一阶导数和二阶导数。

5. 已知函数$f(x) = x^3 + ax^2 + bx + c$在$x = 1$处有极大值，在$x = 3$处有极小值，且$f(0) = 1$，求$a$、$b$、$c$的值。

6. 求函数$f(x) = \frac{x^2}{x + 1}$的导数，并求其在$x = 1$处的切线方程。

7. 求由方程$x^2 + y^2 - 2xy = 0$确定的隐函数$y = y(x)$的导数。

8. 求由参数方程$\begin{cases} x = 1 + t^2 \\ y = t^3 + 2t \end{cases}$确定的函数$y = y(x)$在$t = 1$处的切线方程。

9. 求函数$f(x) = \sin x \cos x$的$n$阶导数。

10. 证明：若函数$f(x)$在$x_0$处可导，则$f(x)$在$x_0$处连续。
