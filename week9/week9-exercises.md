# 第9周：练习题目

## 分部积分法

### 选择题
1. $\int x \cos x dx = $
   A. $x \sin x + \cos x + C$
   B. $x \sin x - \cos x + C$
   C. $-x \sin x + \cos x + C$
   D. $-x \sin x - \cos x + C$

2. $\int \ln x dx = $
   A. $x \ln x + C$
   B. $x \ln x - x + C$
   C. $\frac{(\ln x)^2}{2} + C$
   D. $\frac{1}{x} + C$

3. $\int x^2 e^x dx = $
   A. $x^2 e^x + C$
   B. $(x^2 - 2x + 2)e^x + C$
   C. $(x^2 + 2x + 2)e^x + C$
   D. $(x^2 - 2x)e^x + C$

### 填空题
1. $\int x e^{-x} dx = $ __________

2. $\int \arcsin x dx = $ __________

3. $\int e^x \sin x dx = $ __________

### 解答题
1. 求$\int x \sin 2x dx$。

2. 求$\int x \ln x dx$。

3. 求$\int \arctan x dx$。

4. 求$\int e^x \cos x dx$。

5. 求$\int x^2 \sin x dx$。

6. 求$\int \sqrt{x} \ln x dx$。

7. 求$\int \ln(x + \sqrt{1 + x^2}) dx$。

8. 计算$\int_0^\pi x \sin x dx$。

9. 计算$\int_1^e x \ln x dx$。

10. 计算$\int_0^{\pi/2} e^x \cos x dx$。

## 部分分式分解

### 选择题
1. $\frac{1}{(x - 1)(x - 2)} = $
   A. $\frac{1}{x - 1} + \frac{1}{x - 2}$
   B. $\frac{1}{x - 1} - \frac{1}{x - 2}$
   C. $\frac{1}{x - 2} - \frac{1}{x - 1}$
   D. $\frac{1}{(x - 1)(x - 2)}$

2. $\frac{x}{(x - 1)^2} = $
   A. $\frac{1}{x - 1} + \frac{1}{(x - 1)^2}$
   B. $\frac{1}{x - 1} - \frac{1}{(x - 1)^2}$
   C. $\frac{1}{(x - 1)^2}$
   D. $\frac{x}{(x - 1)^2}$

3. $\frac{1}{x^2 + x - 2} = $
   A. $\frac{1}{3(x - 1)} - \frac{1}{3(x + 2)}$
   B. $\frac{1}{3(x + 1)} - \frac{1}{3(x - 2)}$
   C. $\frac{1}{x - 1} + \frac{1}{x + 2}$
   D. $\frac{1}{x - 1} - \frac{1}{x + 2}$

### 填空题
1. $\frac{1}{x^2 - 4} = $ __________

2. $\frac{x + 1}{x(x - 1)} = $ __________

3. $\frac{2x + 3}{(x - 1)(x^2 + 1)} = $ __________

### 解答题
1. 求$\int \frac{1}{x^2 - 5x + 6} dx$。

2. 求$\int \frac{x}{(x - 1)(x - 2)} dx$。

3. 求$\int \frac{1}{x(x - 1)^2} dx$。

4. 求$\int \frac{x^2 + 1}{x(x^2 - 1)} dx$。

5. 求$\int \frac{1}{x^3 + x} dx$。

6. 求$\int \frac{x}{x^2 + 2x + 2} dx$。

7. 求$\int \frac{x^2}{(x - 1)^3} dx$。

8. 求$\int \frac{1}{x^4 - 1} dx$。

10. 求$\int \frac{e^x}{e^{2x} + 3e^x + 2} dx$。
