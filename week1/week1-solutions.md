### 解答题
1. 已知osθ = 1/2，且θ在第四象限，求inθ和tanθ：
   - 由于  in^2θ + os^2θ = 1 ，所以  in^2θ = 1 - (1/2)^2 = 3/4 。
   - 因为θ在第四象限，所以inθ为负，即inθ = -√3/2。
   - tanθ = inθ/osθ = (-√3/2)/(1/2) = -√3。

2. 证明恒等式：in(A + B) = inA osB + osA inB
   （证明过程略，可参考教材中的证明方法）

3. 求解方程：2in^2x + osx - 1 = 0，其中x ∈ [0, 2i)
   - 利用in^2x = 1 - os^2x，将方程化为：2(1 - os^2x) + osx - 1 = 0
   - 整理得：-2os^2x + osx + 1 = 0，即2os^2x - osx - 1 = 0
   - 令t = osx，方程变为2t^2 - t - 1 = 0，解得t = 1或t = -1/2
   - 当t = 1时，osx = 1，解得x = 0
   - 当t = -1/2时，osx = -1/2，解得x = 2i/3或x = 4i/3
   - 所以方程的解为x = 0, 2i/3, 4i/3

## 综合练习题

1. 函数f(x) = 1/(x^2 + 1)的分析：
   (1) 定义域：全体实数（因为x^2 + 1 > 0对所有实数x成立）
   值域：(0, 1]（当x=0时，f(x)=1；当x趋近于无穷大时，f(x)趋近于0）
   (2) 奇偶性：f(-x) = 1/((-x)^2 + 1) = 1/(x^2 + 1) = f(x)，所以是偶函数
   (3) 单调性：当x < 0时，函数单调递增；当x > 0时，函数单调递减
   (4) 图像形状：关于y轴对称，在x=0处取得最大值1，向两侧逐渐趋近于0，呈“钟形”曲线

2. f(g(x))和g(f(x))的分析：
   - f(g(x)) = in(x^2)，定义域为全体实数，值域为[-1, 1]
   - g(f(x)) = (in x)^2，定义域为全体实数，值域为[0, 1]

3. 三角恒等式化简：
   (1) in^2x + os^2x + tan^2x = 1 + tan^2x = sec^2x
   (2) in2x/(2osx) = (2inxosx)/(2osx) = inx
   (3) os^2x - in^2x = os2x

4. 函数y = 2|x - 1| + 3的图像：
   - 由y = |x|向右平移1个单位，再垂直拉伸为原来的2倍，最后向上平移3个单位得到
   - 图像是一个V形，顶点在(1, 3)，关于直线x=1对称
   - 当x < 1时，函数单调递减；当x > 1时，函数单调递增
   - 最小值为3，无最大值

5. 已知tanθ = 2，求inθ和osθ的值：
   - 由tanθ = inθ/osθ = 2，得inθ = 2osθ
   - 代入in^2θ + os^2θ = 1，得(2osθ)^2 + os^2θ = 1，即5os^2θ = 1，所以osθ = ±1/√5
   - 因为tanθ = 2 > 0，所以θ在第一或第三象限
   - 若θ在第一象限，osθ = 1/√5，inθ = 2/√5
   - 若θ在第三象限，osθ = -1/√5，inθ = -2/√5
