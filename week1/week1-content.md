## 学习内容

### 1. 函数的概念

#### 1.1 函数的定义
- 函数是一种从一个集合（定义域）到另一个集合（值域）的映射
- 每个输入值对应唯一的输出值
- 函数的表示方法：
  - 代数表达式：如  f(x) = x^2 + 1 
  - 图像表示：在坐标系中绘制函数曲线
  - 表格表示：列出输入和对应输出值

#### 1.2 函数的定义域和值域
- 定义域：函数可以接受的所有输入值的集合
- 值域：函数可能产生的所有输出值的集合
- 常见函数的定义域：
  - 多项式函数：全体实数
  - 分式函数：分母不为零
  - 平方根函数：根号内非负
  - 对数函数：真数大于零

#### 1.3 函数的类型
- 线性函数： f(x) = mx + b 
- 二次函数： f(x) = ax^2 + bx + c 
- 幂函数： f(x) = x^n 
- 指数函数： f(x) = a^x （ a > 0, a 
eq 1 ）
- 对数函数： f(x) = og_a x （ a > 0, a 
eq 1 ）
- 三角函数： in x, os x, 	an x  等

### 2. 函数图像

#### 2.1 基本函数图像
- 线性函数：直线
- 二次函数：抛物线
- 幂函数：根据指数不同，图像形状不同
- 指数函数：递增或递减的曲线
- 对数函数：过点 (1, 0) 的曲线
- 三角函数：周期性波动曲线

#### 2.2 图像变换
- 平移变换：
  - 水平平移： f(x - h) ，向右平移  h  个单位
  - 垂直平移： f(x) + k ，向上平移  k  个单位
- 伸缩变换：
  - 水平伸缩： f(cx) ，当  c > 1  时，图像水平压缩
  - 垂直伸缩： c dot f(x) ，当  c > 1  时，图像垂直拉伸
- 反射变换：
  - 关于 y 轴对称： f(-x) 
  - 关于 x 轴对称： -f(x) 

#### 2.3 图像绘制技巧
- 确定定义域和值域
- 找到截距（与坐标轴的交点）
- 确定对称性
- 找到极值点
- 分析函数的单调性
- 确定渐近线
