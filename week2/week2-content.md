# 第二周：极限导论

## 学习目标
- 理解极限的概念和几何意义
- 掌握极限的定义和计算方法
- 理解单侧极限的概念
- 掌握无穷极限和无穷小量的概念
- 学会使用极限运算法则

## 学习内容

### 1. 极限的概念

#### 1.1 数列的极限
- 数列：按照一定规律排列的无穷多个数
- 数列极限的定义：对于数列 {a_n}，如果存在常数A，对于任意给定的正数ε，总存在正整数N，使得当n > N时，|a_n - A| < ε恒成立，则称数列{a_n}收敛于A，记作 lim_{n→∞} a_n = A
- 收敛数列的性质：唯一性、有界性、保号性

#### 1.2 函数的极限
- 函数极限的定义（ε-δ定义）：设函数f(x)在点x_0的某一去心邻域内有定义，如果对于任意给定的正数ε，总存在正数δ，使得当0 < |x - x_0| < δ时，|f(x) - L| < ε恒成立，则称L为函数f(x)当x趋近于x_0时的极限，记作 lim_{x→x_0} f(x) = L
- 函数极限的几何意义：对于任意给定的ε邻域(L-ε, L+ε)，总存在一个δ邻域(x_0-δ, x_0+δ)，使得该δ邻域内的函数值都落在L的ε邻域内

#### 1.3 左极限和右极限
- 左极限： lim_{x→x_0^-} f(x) = L 表示x从左侧趋近于x_0时函数的极限
- 右极限： lim_{x→x_0^+} f(x) = L 表示x从右侧趋近于x_0时函数的极限
- 极限存在的充分必要条件：左极限和右极限都存在且相等

### 2. 无穷极限

#### 2.1 无穷大的定义
- 正无穷大：如果对于任意正数M，总存在正数δ，使得当0 < |x - x_0| < δ时，f(x) > M恒成立，则称f(x)当x→x_0时趋于正无穷大，记作 lim_{x→x_0} f(x) = +∞
- 负无穷大：类似地定义 lim_{x→x_0} f(x) = -∞
- 无穷大不是具体的数值，而是描述函数变化趋势的一种方式

#### 2.2 无穷小量的定义
- 如果 lim_{x→x_0} f(x) = 0，则称f(x)是当x→x_0时的无穷小量
- 无穷小量的性质：
  - 有限个无穷小量的和仍是无穷小量
  - 有限个无穷小量的积仍是无穷小量
  - 常数与无穷小量的积仍是无穷小量
  - 有界函数与无穷小量的积仍是无穷小量

#### 2.3 无穷小量与无穷大量的关系
- 如果 lim f(x) = 0，且f(x)在x_0的邻域内不为零，则 1/f(x) 是无穷大量
- 如果 lim f(x) = ∞，则 1/f(x) 是无穷小量

### 3. 极限的运算法则

#### 3.1 四则运算
如果 lim f(x) = A， lim g(x) = B，则：
- lim [f(x) ± g(x)] = A ± B
- lim [f(x) · g(x)] = A · B
- lim [f(x) / g(x)] = A / B（B ≠ 0）

#### 3.2 复合函数的极限
如果 lim_{x→x_0} g(x) = u_0，且 lim_{u→u_0} f(u) = f(u_0)，则 lim_{x→x_0} f(g(x)) = f(u_0)

#### 3.3 夹逼定理（两边夹定理）
如果函数f(x)、g(x)、h(x)满足：
- 在x_0的某一邻域内（除x_0外）有 g(x) ≤ f(x) ≤ h(x)
- lim g(x) = lim h(x) = L

则 lim f(x) = L

#### 3.4 单调有界原理
- 单调递增且有上界的数列必收敛
- 单调递减且有下界的数列必收敛

### 4. 两个重要极限

#### 4.1 第一个重要极限
$$\lim_{x \to 0} \frac{\sin x}{x} = 1$$

这个极限的证明利用了单位圆和夹逼定理。需要注意的是，这个极限只有在x趋近于0时才能使用，且x必须用弧度表示。

应用示例：
- $\lim_{x \to 0} \frac{\sin 3x}{x} = \lim_{x \to 0} 3 \cdot \frac{\sin 3x}{3x} = 3$
- $\lim_{x \to 0} \frac{\tan x}{x} = \lim_{x \to 0} \frac{\sin x}{x} \cdot \frac{1}{\cos x} = 1$

#### 4.2 第二个重要极限
$$\lim_{x \to \infty} \left(1 + \frac{1}{x}\right)^x = e$$

这个极限也可以写成：
$$\lim_{x \to 0} (1 + x)^{1/x} = e$$

其中e是自然对数的底，约等于2.71828。

应用示例：
- $\lim_{x \to \infty} \left(1 + \frac{2}{x}\right)^x = \lim_{x \to \infty} \left[\left(1 + \frac{2}{x}\right)^{x/2}\right]^2 = e^2$
- $\lim_{x \to 0} (1 + 2x)^{1/x} = \lim_{x \to 0} \left[(1 + 2x)^{1/(2x)}\right]^2 = e^2$

### 5. 求极限的方法

#### 5.1 直接代入法
如果函数在x_0处有定义且连续，则可以直接代入计算极限。

#### 5.2 因式分解法
当直接代入得到0/0型不定式时，可以尝试因式分解，消去零因子。

示例：
$$\lim_{x \to 2} \frac{x^2 - 4}{x - 2} = \lim_{x \to 2} \frac{(x+2)(x-2)}{x-2} = \lim_{x \to 2} (x+2) = 4$$

#### 5.3 有理化法
当分子或分母含有根式时，可以考虑有理化。

示例：
$$\lim_{x \to 0} \frac{\sqrt{1+x} - 1}{x} = \lim_{x \to 0} \frac{(\sqrt{1+x} - 1)(\sqrt{1+x} + 1)}{x(\sqrt{1+x} + 1)} = \lim_{x \to 0} \frac{x}{x(\sqrt{1+x} + 1)} = \frac{1}{2}$$

#### 5.4 等价无穷小替换法
当x→0时，常用的等价无穷小：
- $\sin x \sim x$
- $\tan x \sim x$
- $\arcsin x \sim x$
- $\arctan x \sim x$
- $e^x - 1 \sim x$
- $\ln(1+x) \sim x$
- $(1+x)^\alpha - 1 \sim \alpha x$

#### 5.5 两个重要极限法
灵活运用两个重要极限来求解复杂的极限问题。

### 6. 极限的存在性

#### 6.1 判断极限存在的常用方法
- 直接计算：如果能直接求出极限值，则极限存在
- 夹逼定理：找到两个有相同极限的函数夹逼目标函数
- 单调有界原理：证明数列单调且有界

#### 6.2 极限不存在的情形
- 左右极限不相等
- 函数趋向于无穷大
- 函数在无穷远处振荡
