# 微积分学习项目

## 项目简介

本项目用于系统性学习微积分，基于《普林斯顿微积分读本》教材，提供学习规划、笔记和练习资源。

## 项目结构

- `learning-plan.md` - 详细的学习规划文件，包含19周的学习进度安排
- `notes/` - 学习笔记目录（待创建）
- `exercises/` - 练习题目录（待创建）
- `resources/` - 学习资源目录（待创建）

## 学习资源

- **主要教材**：《普林斯顿微积分读本》
- **辅助资源**：
  - Khan Academy 微积分课程
  - Coursera 上的相关课程
  - GeoGebra 绘图工具
  - 数学软件（如Mathematica、MATLAB）

## 学习规划

学习规划基于《普林斯顿微积分读本》的内容，分为以下阶段：

1. **预备知识**（1周）：函数、图像、三角学复习
2. **微分学基础**（5周）：极限、导数、导数应用等
3. **积分学基础**（5周）：积分、换元积分、分部积分等
4. **级数**（3周）：级数收敛性、幂级数、泰勒级数等
5. **多元微积分**（3周）：多元函数、偏导数、多重积分等
6. **复习与巩固**（2周）：综合复习和模拟测试

详细的学习计划请参考 `learning-plan.md` 文件。

## 使用方法

1. 按照 `learning-plan.md` 中的进度安排进行学习
2. 在 `notes/` 目录中记录学习笔记
3. 在 `exercises/` 目录中存放练习题和解答
4. 在 `resources/` 目录中整理学习资源

## 学习建议

- 每天投入2-3小时学习微积分
- 完成教材中80%以上的习题
- 定期复习之前学习的内容
- 遇到疑难问题时参考网络资源或向学习社区求助

## 评估方式

- 每周完成一套本周学习内容的测试题
- 每个阶段结束后进行一次综合测试
- 完成一套完整的微积分测试题，评估整体学习成果
