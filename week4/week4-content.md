# 第四周：导数的应用

## 学习目标
- 理解函数单调性与导数的关系
- 掌握函数极值的判定方法
- 理解函数的凹凸性与二阶导数的关系
- 掌握函数图像的绘制方法
- 学会应用导数解决实际优化问题

## 学习内容

### 1. 函数的单调性与导数

#### 1.1 单调性的判定定理
- 设函数 $y = f(x)$ 在区间 $I$ 上可导，则：
  - 若在区间 $I$ 上 $f'(x) > 0$，则 $f(x)$ 在 $I$ 上单调递增
  - 若在区间 $I$ 上 $f'(x) < 0$，则 $f(x)$ 在 $I$ 上单调递减
  - 若在区间 $I$ 上 $f'(x) = 0$，则 $f(x)$ 在 $I$ 上为常数函数

#### 1.2 单调区间的划分
- 步骤：
  1. 确定函数的定义域
  2. 求导数 $f'(x)$
  3. 找出 $f'(x) = 0$ 的点和 $f'(x)$ 不存在的点
  4. 用这些点将定义域分成若干区间
  5. 在每个区间内判断 $f'(x)$ 的符号，确定单调性

#### 1.3 示例
- 分析函数 $f(x) = x^3 - 3x$ 的单调性
  - 定义域：$(-\infty, +\infty)$
  - 导数：$f'(x) = 3x^2 - 3 = 3(x-1)(x+1)$
  - 临界点：$x = -1$ 和 $x = 1$
  - 单调区间：
    - $(-\infty, -1)$：$f'(x) > 0$，单调递增
    - $(-1, 1)$：$f'(x) < 0$，单调递减
    - $(1, +\infty)$：$f'(x) > 0$，单调递增

### 2. 函数的极值

#### 2.1 极值的定义
- 极大值：如果存在点 $x_0$ 的某个邻域，使得对该邻域内所有 $x$ 都有 $f(x) \leq f(x_0)$，则称 $f(x_0)$ 为极大值
- 极小值：如果存在点 $x_0$ 的某个邻域，使得对该邻域内所有 $x$ 都有 $f(x) \geq f(x_0)$，则称 $f(x_0)$ 为极小值
- 极值：极大值和极小值统称为极值

#### 2.2 费马定理（必要条件）
- 若函数 $f(x)$ 在点 $x_0$ 处可导，且在 $x_0$ 处取得极值，则 $f'(x_0) = 0$
- 驻点：使 $f'(x) = 0$ 的点称为驻点
- 注意：可导函数的极值点一定是驻点，但驻点不一定是极值点

#### 2.3 第一充分条件
- 设函数 $f(x)$ 在点 $x_0$ 的某去心邻域内可导，且 $f'(x_0) = 0$（或 $f(x)$ 在 $x_0$ 处连续但不可导）：
  1. 当 $x$ 从左侧趋近于 $x_0$ 时 $f'(x) > 0$，从右侧趋近于 $x_0$ 时 $f'(x) < 0$，则 $f(x_0)$ 是极大值
  2. 当 $x$ 从左侧趋近于 $x_0$ 时 $f'(x) < 0$，从右侧趋近于 $x_0$ 时 $f'(x) > 0$，则 $f(x_0)$ 是极小值
  3. 当 $x$ 从左右两侧趋近于 $x_0$ 时，$f'(x)$ 符号不变，则 $f(x_0)$ 不是极值

#### 2.4 第二充分条件
- 设函数 $f(x)$ 在点 $x_0$ 处具有二阶导数，且 $f'(x_0) = 0$，$f''(x_0) \neq 0$：
  1. 若 $f''(x_0) < 0$，则 $f(x)$ 在 $x_0$ 处取得极大值
  2. 若 $f''(x_0) > 0$，则 $f(x)$ 在 $x_0$ 处取得极小值
- 注意：当 $f''(x_0) = 0$ 时，此方法失效，需使用第一充分条件

#### 2.5 求函数极值的步骤
1. 确定函数的定义域
2. 求导数 $f'(x)$
3. 找出所有驻点和不可导点
4. 利用第一或第二充分条件判断这些点是否为极值点
5. 求出极值

#### 2.6 示例
- 求函数 $f(x) = x^3 - 3x$ 的极值
  - 定义域：$(-\infty, +\infty)$
  - 导数：$f'(x) = 3x^2 - 3 = 3(x-1)(x+1)$
  - 驻点：$x = -1$ 和 $x = 1$
  - 二阶导数：$f''(x) = 6x$
  - 利用第二充分条件：
    - $f''(-1) = -6 < 0$，所以 $f(-1) = 2$ 是极大值
    - $f''(1) = 6 > 0$，所以 $f(1) = -2$ 是极小值

### 3. 函数的最大值和最小值

#### 3.1 闭区间上的最大值和最小值
- 若函数 $f(x)$ 在闭区间 $[a, b]$ 上连续，则 $f(x)$ 在 $[a, b]$ 上必有最大值和最小值
- 求法：
  1. 求出 $f(x)$ 在 $(a, b)$ 内的所有驻点和不可导点
  2. 计算这些点的函数值以及端点的函数值 $f(a)$ 和 $f(b)$
  3. 比较这些值，最大的就是最大值，最小的就是最小值

#### 3.2 开区间或无穷区间上的最值
- 如果函数在区间内只有一个极值点，则该极值点就是最值点
- 若函数在区间内单调，则最值在端点处取得

#### 3.3 示例
- 求函数 $f(x) = x^3 - 3x$ 在 $[-2, 2]$ 上的最大值和最小值
  - 驻点：$x = -1$ 和 $x = 1$
  - 计算各点函数值：
    - $f(-2) = (-2)^3 - 3(-2) = -8 + 6 = -2$
    - $f(-1) = (-1)^3 - 3(-1) = -1 + 3 = 2$
    - $f(1) = 1^3 - 3(1) = 1 - 3 = -2$
    - $f(2) = 2^3 - 3(2) = 8 - 6 = 2$
  - 最大值：$2$（在 $x = -1$ 和 $x = 2$ 处取得）
  - 最小值：$-2$（在 $x = 1$ 和 $x = -2$ 处取得）

### 4. 函数的凹凸性

#### 4.1 凹凸性的定义
- 设函数 $f(x)$ 在区间 $I$ 上连续：
  - 若对任意 $x_1, x_2 \in I$，恒有 $f\left(\frac{x_1+x_2}{2}\right) < \frac{f(x_1)+f(x_2)}{2}$，则称 $f(x)$ 在 $I$ 上是凹函数（上凹）
  - 若对任意 $x_1, x_2 \in I$，恒有 $f\left(\frac{x_1+x_2}{2}\right) > \frac{f(x_1)+f(x_2)}{2}$，则称 $f(x)$ 在 $I$ 上是凸函数（下凹）

#### 4.2 凹凸性的判定定理
- 设函数 $f(x)$ 在区间 $I$ 上具有二阶导数：
  - 若在 $I$ 上 $f''(x) > 0$，则 $f(x)$ 在 $I$ 上是凹函数
  - 若在 $I$ 上 $f''(x) < 0$，则 $f(x)$ 在 $I$ 上是凸函数

#### 4.3 拐点的定义
- 连续曲线 $y = f(x)$ 上凹弧与凸弧的分界点称为曲线的拐点

#### 4.4 拐点的判定
- 求拐点的步骤：
  1. 求 $f''(x)$
  2. 找出使 $f''(x) = 0$ 的点和 $f''(x)$ 不存在的点
  3. 检查这些点的左右两侧 $f''(x)$ 的符号，若符号改变，则该点是拐点

#### 4.5 示例
- 讨论函数 $f(x) = x^3 - 3x$ 的凹凸性和拐点
  - 一阶导数：$f'(x) = 3x^2 - 3$
  - 二阶导数：$f''(x) = 6x$
  - 令 $f''(x) = 0$，得 $x = 0$
  - 凹凸性：
    - 当 $x < 0$ 时，$f''(x) < 0$，函数是凸函数
    - 当 $x > 0$ 时，$f''(x) > 0$，函数是凹函数
  - 拐点：$(0, 0)$

### 5. 函数图像的绘制

#### 5.1 绘制函数图像的一般步骤
1. 确定函数的定义域、值域
2. 考察函数的奇偶性、周期性
3. 求函数的一阶导数，确定单调区间和极值点
4. 求函数的二阶导数，确定凹凸区间和拐点
5. 确定函数的渐近线（如果有）
6. 计算一些特殊点的函数值（如与坐标轴的交点等）
7. 综合以上信息，绘制函数图像

#### 5.2 渐近线的求法
- 垂直渐近线：若 $\lim_{x \to x_0} f(x) = \infty$，则 $x = x_0$ 是垂直渐近线
- 水平渐近线：若 $\lim_{x \to \pm\infty} f(x) = L$，则 $y = L$ 是水平渐近线
- 斜渐近线：若 $\lim_{x \to \pm\infty} [f(x) - (ax + b)] = 0$，其中 $a \neq 0$，则 $y = ax + b$ 是斜渐近线，其中 $a = \lim_{x \to \pm\infty} \frac{f(x)}{x}$，$b = \lim_{x \to \pm\infty} [f(x) - ax]$

#### 5.3 示例
- 绘制函数 $f(x) = x^3 - 3x$ 的图像
  - 定义域：$(-\infty, +\infty)$
  - 奇偶性：奇函数，关于原点对称
  - 单调性：
    - $(-\infty, -1)$：单调递增
    - $(-1, 1)$：单调递减
    - $(1, +\infty)$：单调递增
  - 极值：
    - 极大值 $f(-1) = 2$
    - 极小值 $f(1) = -2$
  - 凹凸性：
    - $(-\infty, 0)$：凸函数
    - $(0, +\infty)$：凹函数
  - 拐点：$(0, 0)$
  - 与坐标轴交点：$(0, 0)$，$(-\sqrt{3}, 0)$，$(\sqrt{3}, 0)$
  - 渐近线：无

### 6. 导数在实际问题中的应用

#### 6.1 优化问题的一般步骤
1. 建立目标函数
2. 确定自变量的取值范围
3. 求目标函数的极值和最值
4. 验证解的合理性并回答问题

#### 6.2 几何应用示例
- 问题：用一张长为 $8$、宽为 $5$ 的矩形铁皮，在四个角上剪去一个边长为 $x$ 的小正方形，然后将四边折起做成一个无盖的盒子。问 $x$ 为何值时，盒子的容积最大？
  - 建立函数：盒子的长为 $8-2x$，宽为 $5-2x$，高为 $x$，所以容积为
    $$V(x) = x(8-2x)(5-2x) = 4x^3 - 26x^2 + 40x$$
  - 定义域：$0 < x < \frac{5}{2}$
  - 求导：$V'(x) = 12x^2 - 52x + 40 = 4(3x^2 - 13x + 10) = 4(3x-10)(x-1)$
  - 驻点：令 $V'(x) = 0$，得 $x = 1$（$x = \frac{10}{3}$ 超出定义域）
  - 二阶导数：$V''(x) = 24x - 52$，$V''(1) = -28 < 0$，所以 $x = 1$ 时取得极大值
  - 结论：当 $x = 1$ 时，盒子的容积最大

#### 6.3 经济应用示例
- 问题：设某工厂生产 $x$ 件产品的成本为 $C(x) = 5000 + 200x + \frac{1}{40}x^2$（元），售价为 $P(x) = 400 - \frac{1}{20}x$（元/件）。问生产多少件产品时利润最大？
  - 利润函数：$L(x) = R(x) - C(x)$，其中收入函数 $R(x) = xP(x) = x(400 - \frac{1}{20}x) = 400x - \frac{1}{20}x^2$
  - 所以 $L(x) = (400x - \frac{1}{20}x^2) - (5000 + 200x + \frac{1}{40}x^2) = -5000 + 200x - \frac{3}{40}x^2$
  - 求导：$L'(x) = 200 - \frac{3}{20}x$
  - 令 $L'(x) = 0$，得 $x = \frac{4000}{3} \approx 1333.33$
  - 二阶导数：$L''(x) = -\frac{3}{20} < 0$，所以这是最大值点
  - 结论：生产约 1333 件产品时利润最大

#### 6.4 物理应用示例
- 问题：从半径为 $R$ 的圆铁片上挖去一个扇形，将剩下的部分做成一个圆锥形容器。问扇形的圆心角 $\alpha$ 为多大时，圆锥的容积最大？
  - 分析：设圆锥的底面半径为 $r$，高为 $h$，则 $r^2 + h^2 = R^2$
  - 剩余弧长：$(2\pi - \alpha)R = 2\pi r$，所以 $r = R(1 - \frac{\alpha}{2\pi})$
  - 高：$h = \sqrt{R^2 - r^2} = R\sqrt{\frac{\alpha}{\pi} - \frac{\alpha^2}{4\pi^2}}$
  - 容积：$V(\alpha) = \frac{1}{3}\pi r^2 h$
  - 为简化计算，令 $f(\alpha) = r^4 h^2$，求 $f(\alpha)$ 的最大值
  - 计算得，当 $\alpha = 2\pi(1 - \frac{\sqrt{6}}{3}) \approx 1.15\pi$ 时，容积最大
