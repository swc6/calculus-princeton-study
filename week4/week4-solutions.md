# 第四周练习题答案

## 一、基础题

### 1. 函数的单调性

**(1) $f(x) = 2x^3 - 9x^2 + 12x - 3$**

解：定义域为 $(-\infty, +\infty)$
$f'(x) = 6x^2 - 18x + 12 = 6(x-1)(x-2)$
令 $f'(x) = 0$，得驻点 $x=1, x=2$
单调区间：
- $(-\infty, 1)$：$f'(x) > 0$，单调递增
- $(1, 2)$：$f'(x) < 0$，单调递减
- $(2, +\infty)$：$f'(x) > 0$，单调递增

**(2) $f(x) = x - \ln(1+x)$**

解：定义域为 $(-1, +\infty)$
$f'(x) = 1 - \frac{1}{1+x} = \frac{x}{1+x}$
令 $f'(x) = 0$，得驻点 $x=0$
单调区间：
- $(-1, 0)$：$f'(x) < 0$，单调递减
- $(0, +\infty)$：$f'(x) > 0$，单调递增

**(3) $f(x) = \frac{x^2 - 2x + 2}{x - 1}$**

解：定义域为 $(-\infty, 1) \cup (1, +\infty)$
化简：$f(x) = \frac{(x-1)^2 + 1}{x-1} = (x-1) + \frac{1}{x-1}$
$f'(x) = 1 - \frac{1}{(x-1)^2} = \frac{(x-1)^2 - 1}{(x-1)^2} = \frac{x(x-2)}{(x-1)^2}$
令 $f'(x) = 0$，得驻点 $x=0, x=2$
单调区间：
- $(-\infty, 0)$：$f'(x) > 0$，单调递增
- $(0, 1)$：$f'(x) < 0$，单调递减
- $(1, 2)$：$f'(x) < 0$，单调递减
- $(2, +\infty)$：$f'(x) > 0$，单调递增

### 2. 函数的极值

**(1) $f(x) = x^3 - 3x^2 - 9x + 5$**

解：$f'(x) = 3x^2 - 6x - 9 = 3(x-3)(x+1)$
令 $f'(x) = 0$，得驻点 $x=-1, x=3$
$f''(x) = 6x - 6$
$f''(-1) = -12 < 0$，所以 $f(-1) = 10 是极大值
$f''(3) = 12 > 0$，所以 $f(3) = -22 是极小值

**(2) $f(x) = x - \frac{3}{2}x^{\frac{2}{3}}$**

解：定义域为 $(-\infty, +\infty)$
$f'(x) = 1 - x^{-\frac{1}{3}} = 1 - \frac{1}{\sqrt[3]{x}} = \frac{\sqrt[3]{x} - 1}{\sqrt[3]{x}}$
令 $f'(x) = 0$，得驻点 $x=1$；当 $x=0$ 时 $f'(x)$ 不存在
用第一充分条件判断：
- $x < 0$ 时，$f'(x) > 0$
- $0 < x < 1$ 时，$f'(x) < 0$
- $x > 1$ 时，$f'(x) > 0$
所以：
- $f(0) = 0$ 是极大值
- $f(1) = -\frac{1}{2}$ 是极小值

**(3) $f(x) = \frac{\ln x}{x}$**

解：定义域为 $(0, +\infty)$
$f'(x) = \frac{\frac{1}{x} \cdot x - \ln x \cdot 1}{x^2} = \frac{1 - \ln x}{x^2}$
令 $f'(x) = 0$，得驻点 $x=e$
当 $0 < x < e$ 时，$f'(x) > 0$
当 $x > e$ 时，$f'(x) < 0$
所以 $f(e) = \frac{1}{e}$ 是极大值

### 3. 函数的最值

**(1) $f(x) = 2x^3 - 3x^2 - 12x + 1$，$x \in [-2, 3]$**

解：$f'(x) = 6x^2 - 6x - 12 = 6(x-2)(x+1)$
驻点：$x=-1, x=2$
计算各点函数值：
$f(-2) = -3$，$f(-1) = 8$，$f(2) = -19$，$f(3) = -8$
最大值：$8$（在 $x=-1$ 处）
最小值：$-19$（在 $x=2$ 处）

**(2) $f(x) = x + 2\sqrt{x}$，$x \in [0, 4]$**

解：$f'(x) = 1 + \frac{1}{\sqrt{x}} > 0$ 对所有 $x > 0$ 成立
函数在 $[0, 4]$ 上单调递增
$f(0) = 0$，$f(4) = 8$
最大值：$8$（在 $x=4$ 处）
最小值：$0$（在 $x=0$ 处）

**(3) $f(x) = \ln(x^2 + 1)$，$x \in [-1, 2]$**

解：$f'(x) = \frac{2x}{x^2 + 1}$
令 $f'(x) = 0$，得驻点 $x=0$
计算各点函数值：
$f(-1) = \ln 2$，$f(0) = 0$，$f(2) = \ln 5$
最大值：$\ln 5$（在 $x=2$ 处）
最小值：$0$（在 $x=0$ 处）

### 4. 函数的凹凸性与拐点

**(1) $f(x) = x^3 - 5x^2 + 3x - 5$**

解：$f'(x) = 3x^2 - 10x + 3$
$f''(x) = 6x - 10$
令 $f''(x) = 0$，得 $x = \frac{5}{3}$
当 $x < \frac{5}{3}$ 时，$f''(x) < 0$，凸函数
当 $x > \frac{5}{3}$ 时，$f''(x) > 0$，凹函数
拐点：$(\frac{5}{3}, -\frac{250}{27})$

**(2) $f(x) = x e^{-x}$**

解：$f'(x) = e^{-x} - x e^{-x} = e^{-x}(1 - x)$
$f''(x) = -e^{-x}(1 - x) - e^{-x} = e^{-x}(x - 2)$
令 $f''(x) = 0$，得 $x = 2$
当 $x < 2$ 时，$f''(x) < 0$，凸函数
当 $x > 2$ 时，$f''(x) > 0$，凹函数
拐点：$(2, \frac{2}{e^2})$

**(3) $f(x) = \ln(1 + x^2)$**

解：$f'(x) = \frac{2x}{1 + x^2}$
$f''(x) = \frac{2(1 + x^2) - 2x \cdot 2x}{(1 + x^2)^2} = \frac{2(1 - x^2)}{(1 + x^2)^2}$
令 $f''(x) = 0$，得 $x = \pm 1$
凹凸性：
- $x < -1$：$f''(x) < 0$，凸函数
- $-1 < x < 1$：$f''(x) > 0$，凹函数
- $x > 1$：$f''(x) < 0$，凸函数
拐点：$(-1, \ln 2)$ 和 $(1, \ln 2)$

## 二、中级题

### 5. 函数图像绘制

**(1) $f(x) = x^4 - 2x^2 + 3$**

解：
- 定义域：$(-\infty, +\infty)$
- 奇偶性：偶函数
- $f'(x) = 4x^3 - 4x = 4x(x^2 - 1) = 4x(x-1)(x+1)$
- 驻点：$x=-1, x=0, x=1$
- $f''(x) = 12x^2 - 4 = 4(3x^2 - 1)$
- 令 $f''(x) = 0$，得 $x = \pm \frac{1}{\sqrt{3}}$
- 单调性：
  - $(-\infty, -1)$：递减
  - $(-1, 0)$：递增
  - $(0, 1)$：递减
  - $(1, +\infty)$：递增
- 极值：
  - 极小值 $f(-1) = 2$
  - 极大值 $f(0) = 3$
  - 极小值 $f(1) = 2$
- 凹凸性：
  - $(-\infty, -\frac{1}{\sqrt{3}})$：凹
  - $(-\frac{1}{\sqrt{3}}, \frac{1}{\sqrt{3}})$：凸
  - $(\frac{1}{\sqrt{3}}, +\infty)$：凹
- 拐点：$(\pm \frac{1}{\sqrt{3}}, \frac{22}{9})$

**(2) $f(x) = \frac{x}{1 + x^2}$**

解：
- 定义域：$(-\infty, +\infty)$
- 奇偶性：奇函数
- $f'(x) = \frac{(1 + x^2) - x \cdot 2x}{(1 + x^2)^2} = \frac{1 - x^2}{(1 + x^2)^2}$
- 驻点：$x = \pm 1$
- $f''(x) = \frac{-2x(1 + x^2)^2 - (1 - x^2) \cdot 2(1 + x^2) \cdot 2x}{(1 + x^2)^4} = \frac{2x(x^2 - 3)}{(1 + x^2)^3}$
- 令 $f''(x) = 0$，得 $x = 0, \pm \sqrt{3}$
- 单调性：
  - $(-\infty, -1)$：递减
  - $(-1, 1)$：递增
  - $(1, +\infty)$：递减
- 极值：
  - 极小值 $f(-1) = -\frac{1}{2}$
  - 极大值 $f(1) = \frac{1}{2}$
- 水平渐近线：$\lim_{x \to \pm\infty} f(x) = 0$，所以 $y=0$
- 拐点：$(-\sqrt{3}, -\frac{\sqrt{3}}{4}), (0, 0), (\sqrt{3}, \frac{\sqrt{3}}{4})$

**(3) $f(x) = e^{-x^2}$（高斯曲线）**

解：
- 定义域：$(-\infty, +\infty)$
- 奇偶性：偶函数
- $f'(x) = -2x e^{-x^2}$
- 驻点：$x = 0$
- $f''(x) = -2e^{-x^2} + 4x^2 e^{-x^2} = 2e^{-x^2}(2x^2 - 1)$
- 令 $f''(x) = 0$，得 $x = \pm \frac{1}{\sqrt{2}}$
- 单调性：
  - $(-\infty, 0)$：递增
  - $(0, +\infty)$：递减
- 极值：$f(0) = 1$（极大值）
- 水平渐近线：$\lim_{x \to \pm\infty} f(x) = 0$，所以 $y=0$
- 拐点：$(\pm \frac{1}{\sqrt{2}}, e^{-\frac{1}{2}})$

### 6. 证明题

**(1) 当 $x > 0$ 时，$1 + \frac{1}{2}x > \sqrt{1 + x}$**

证：令 $f(x) = 1 + \frac{1}{2}x - \sqrt{1 + x}$
$f(0) = 0$
$f'(x) = \frac{1}{2} - \frac{1}{2\sqrt{1 + x}} = \frac{\sqrt{1 + x} - 1}{2\sqrt{1 + x}}$
当 $x > 0$ 时，$f'(x) > 0$，所以 $f(x)$ 在 $[0, +\infty)$ 上单调递增
故当 $x > 0$ 时，$f(x) > f(0) = 0$，即 $1 + \frac{1}{2}x > \sqrt{1 + x}$

**(2) 当 $x > 1$ 时，$2\sqrt{x} > 3 - \frac{1}{x}$**

证：令 $f(x) = 2\sqrt{x} - 3 + \frac{1}{x}$
$f(1) = 0$
$f'(x) = \frac{1}{\sqrt{x}} - \frac{1}{x^2} = \frac{x^{3/2} - 1}{x^2}$
当 $x > 1$ 时，$f'(x) > 0$，所以 $f(x)$ 在 $[1, +\infty)$ 上单调递增
故当 $x > 1$ 时，$f(x) > f(1) = 0$，即 $2\sqrt{x} > 3 - \frac{1}{x}$

**(3) 当 $x > 0$ 时，$\ln(1 + x) > \frac{\arctan x}{1 + x}$**

证：令 $f(x) = (1 + x)\ln(1 + x) - \arctan x$
$f(0) = 0$
$f'(x) = \ln(1 + x) + 1 - \frac{1}{1 + x^2}$
$f'(0) = 0$
$f''(x) = \frac{1}{1 + x} + \frac{2x}{(1 + x^2)^2}$
当 $x > 0$ 时，$f''(x) > 0$，所以 $f'(x)$ 在 $[0, +\infty)$ 上单调递增
故当 $x > 0$ 时，$f'(x) > f'(0) = 0$，进而 $f(x)$ 在 $[0, +\infty)$ 上单调递增
所以当 $x > 0$ 时，$f(x) > f(0) = 0$，即 $(1 + x)\ln(1 + x) > \arctan x$，即 $\ln(1 + x) > \frac{\arctan x}{1 + x}$

### 7. 方程实根的个数

**(1) $x^3 - 6x^2 + 9x - 10 = 0$**

解：令 $f(x) = x^3 - 6x^2 + 9x - 10$
$f'(x) = 3x^2 - 12x + 9 = 3(x-1)(x-3)$
驻点：$x=1, x=3$
$f(1) = -6$，$f(3) = -10$
当 $x < 1$ 时，$f'(x) > 0$，单调递增
当 $1 < x < 3$ 时，$f'(x) < 0$，单调递减
当 $x > 3$ 时，$f'(x) > 0$，单调递增
$\lim_{x \to -\infty} f(x) = -\infty$，$\lim_{x \to +\infty} f(x) = +\infty$
所以方程只有一个实根，在 $(3, +\infty)$ 内

**(2) $e^x - ax = 0$（$a > 0$）**

解：令 $f(x) = e^x - ax$
$f'(x) = e^x - a$
驻点：$x = \ln a$
$f''(x) = e^x > 0$，所以 $f(x)$ 在 $x = \ln a$ 处取得极小值 $f(\ln a) = a - a \ln a = a(1 - \ln a)$
讨论：
- 当 $a(1 - \ln a) > 0$ 即 $0 < a < e$ 时，$f(x) > 0$ 恒成立，方程无实根
- 当 $a(1 - \ln a) = 0$ 即 $a = e$ 时，方程有一个实根
- 当 $a(1 - \ln a) < 0$ 即 $a > e$ 时，方程有两个实根

## 三、应用题

### 8. 几何应用

**(1) 一个圆柱形有盖容器的容积为 $V$，问底半径 $r$ 和高 $h$ 各为多少时，用料最省？**

解：表面积 $S = 2\pi r^2 + 2\pi r h$
由 $V = \pi r^2 h$，得 $h = \frac{V}{\pi r^2}$
代入得 $S(r) = 2\pi r^2 + \frac{2V}{r}$
$S'(r) = 4\pi r - \frac{2V}{r^2}$
令 $S'(r) = 0$，得 $4\pi r = \frac{2V}{r^2}$，即 $r^3 = \frac{V}{2\pi}$，$r = \sqrt[3]{\frac{V}{2\pi}}$
此时 $h = \frac{V}{\pi r^2} = 2r$
所以当 $h = 2r$ 时，用料最省

**(2) 从一块边长为 $a$ 的正方形铁片的四个角上，各剪去一个小正方形，然后将四边折起做成一个无盖的方盒。问剪去的小正方形边长为多少时，方盒的容积最大？**

解：设剪去的小正方形边长为 $x$，则方盒的容积为
$V(x) = x(a - 2x)^2$，$0 < x < \frac{a}{2}$
$V'(x) = (a - 2x)^2 - 4x(a - 2x) = (a - 2x)(a - 6x)$
令 $V'(x) = 0$，得 $x = \frac{a}{6}$（$x = \frac{a}{2}$ 超出定义域）
$V''(x) = -2(a - 6x) - 6(a - 2x) = 24x - 8a$
$V''(\frac{a}{6}) = -4a < 0$，所以 $x = \frac{a}{6}$ 时容积最大
即剪去的小正方形边长为 $\frac{a}{6}$ 时，方盒的容积最大

**(3) 在椭圆 $\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$ 的第一象限部分上求一点，使椭圆在该点的切线与两坐标轴围成的三角形面积最小。**

解：设切点为 $(x_0, y_0)$，则切线方程为 $\frac{x_0 x}{a^2} + \frac{y_0 y}{b^2} = 1$
切线在 $x$ 轴上的截距为 $\frac{a^2}{x_0}$，在 $y$ 轴上的截距为 $\frac{b^2}{y_0}$
三角形面积 $S = \frac{1}{2} \cdot \frac{a^2}{x_0} \cdot \frac{b^2}{y_0} = \frac{a^2 b^2}{2 x_0 y_0}$
由于 $\frac{x_0^2}{a^2} + \frac{y_0^2}{b^2} = 1$，所以 $y_0 = \frac{b}{a} \sqrt{a^2 - x_0^2}$
代入得 $S(x_0) = \frac{a^3 b}{2 x_0 \sqrt{a^2 - x_0^2}}$
为简化计算，求 $f(x_0) = x_0^2(a^2 - x_0^2)$ 的最大值
$f'(x_0) = 2x_0(a^2 - x_0^2) - 2x_0^3 = 2a^2 x_0 - 4x_0^3$
令 $f'(x_0) = 0$，得 $x_0 = \frac{a}{\sqrt{2}}$
此时 $y_0 = \frac{b}{\sqrt{2}}$
所以切点为 $(\frac{a}{\sqrt{2}}, \frac{b}{\sqrt{2}})$ 时，三角形面积最小

### 9. 物理应用

**(1) 设有长度为 $l$ 的线段 $AB$，在 $A$ 和 $B$ 处分别有力 $F_1$ 和 $F_2$ 作用。问在 $AB$ 上何处放置支点，才能使线段平衡？**

解：设支点距离 $A$ 为 $x$，则距离 $B$ 为 $l - x$
力矩平衡条件：$F_1 x = F_2 (l - x)$
解得 $x = \frac{F_2 l}{F_1 + F_2}$
所以支点应放置在距离 $A$ 为 $\frac{F_2 l}{F_1 + F_2}$ 处

**(2) 设有重量为 $5kg$ 的物体，置于水平面上，受力 $F$ 的作用而开始移动。已知摩擦系数为 $0.25$，问力 $F$ 与水平线的夹角 $\alpha$ 为多少时，力 $F$ 最小？**

解：设力 $F$ 与水平线的夹角为 $\alpha$，则
水平分力：$F \cos \alpha$
垂直分力：$F \sin \alpha$
正压力：$N = mg - F \sin \alpha = 5g - F \sin \alpha$
摩擦力：$f = \mu N = 0.25(5g - F \sin \alpha)$
平衡条件：$F \cos \alpha = f$
即 $F \cos \alpha = 0.25(5g - F \sin \alpha)$
解得 $F(\alpha) = \frac{1.25g}{\cos \alpha + 0.25 \sin \alpha}$
求 $F(\alpha)$ 的最小值，等价于求 $g(\alpha) = \cos \alpha + 0.25 \sin \alpha$ 的最大值
$g'(\alpha) = -\sin \alpha + 0.25 \cos \alpha$
令 $g'(\alpha) = 0$，得 $\tan \alpha = 0.25$，即 $\alpha = \arctan 0.25$
所以当 $\alpha = \arctan 0.25$ 时，力 $F$ 最小

**(3) 一观察者站在离塔基 $100$ 米的地方，塔高为 $100$ 米。问观察者的眼睛离地面多高时，观察塔顶的仰角最大？**

解：设观察者的眼睛离地面 $h$ 米，则
仰角 $\theta = \arctan \frac{100 - h}{100} - \arctan \frac{-h}{100}$（注意：应为 $\theta = \arctan \frac{100 - h}{100} - \arctan \frac{0 - h}{100} = \arctan \frac{100 - h}{100} + \arctan \frac{h}{100}$）
令 $x = \frac{h}{100}$，则 $\theta = \arctan(1 - x) + \arctan x$
$\tan \theta = \tan[\arctan(1 - x) + \arctan x] = \frac{(1 - x) + x}{1 - (1 - x)x} = \frac{1}{1 - x + x^2}$
求 $\tan \theta$ 的最大值，即求分母 $1 - x + x^2$ 的最小值
令 $f(x) = x^2 - x + 1$，$f'(x) = 2x - 1$
令 $f'(x) = 0$，得 $x = \frac{1}{2}$
所以 $h = 50$ 米时，仰角最大

### 10. 经济应用

**(1) 设某商品的成本函数为 $C(x) = 1000 + 2x + 0.001x^2$（元），其中 $x$ 为产量。商品的单价为 $P(x) = 5 - 0.001x$（元/件）。问产量为多少时，利润最大？最大利润是多少？**

解：收入函数 $R(x) = xP(x) = 5x - 0.001x^2$
利润函数 $L(x) = R(x) - C(x) = -1000 + 3x - 0.002x^2$
$L'(x) = 3 - 0.004x$
令 $L'(x) = 0$，得 $x = 750$
$L''(x) = -0.004 < 0$，所以 $x = 750$ 时利润最大
最大利润 $L(750) = 125$ 元

**(2) 某工厂生产某产品，固定成本为 $20000$ 元，每生产一单位产品，成本增加 $100$ 元。已知总收入 $R$ 是年产量 $x$ 的函数：$R(x) = \begin{cases} 400x - \frac{1}{2}x^2, & 0 \leq x \leq 400 \\ 80000, & x > 400 \end{cases}$。问每年生产多少单位产品时，利润最大？最大利润是多少？**

解：成本函数 $C(x) = 20000 + 100x$
利润函数：
当 $0 \leq x \leq 400$ 时，$L(x) = 300x - \frac{1}{2}x^2 - 20000$
$L'(x) = 300 - x$
令 $L'(x) = 0$，得 $x = 300$
$L(300) = 25000$ 元
当 $x > 400$ 时，$L(x) = 60000 - 100x$，单调递减
所以当 $x = 300$ 时，利润最大，最大利润为 $25000$ 元

**(3) 设某商品的需求函数为 $Q = 100 - 5P$，其中 $Q$ 为需求量，$P$ 为价格。问价格为多少时，总收益最大？最大收益是多少？**

解：由 $Q = 100 - 5P$，得 $P = 20 - \frac{Q}{5}$
总收益 $R(Q) = QP = 20Q - \frac{Q^2}{5}$
$R'(Q) = 20 - \frac{2Q}{5}$
令 $R'(Q) = 0$，得 $Q = 50$
此时 $P = 10$
最大收益 $R(50) = 500$
所以当价格为 $10$ 元时，总收益最大，最大收益为 $500$ 元

## 四、综合题

### 11. 综合题

**(1) 设函数 $f(x)$ 在 $[0, 1]$ 上可导，且 $0 < f(x) < 1$，$f'(x) \neq 1$。证明：方程 $f(x) = x$ 在 $(0, 1)$ 内有唯一实根。**

证：令 $g(x) = f(x) - x$
则 $g(0) = f(0) > 0$，$g(1) = f(1) - 1 < 0$
由零点定理，存在 $\xi \in (0, 1)$，使得 $g(\xi) = 0$，即 $f(\xi) = \xi$
再证唯一性：假设存在两个点 $\xi_1, \xi_2 \in (0, 1)$，使得 $f(\xi_1) = \xi_1$，$f(\xi_2) = \xi_2$，且 $\xi_1 < \xi_2$
由罗尔定理，存在 $\eta \in (\xi_1, \xi_2)$，使得 $g'(\eta) = 0$，即 $f'(\eta) = 1$，与题设矛盾
所以方程 $f(x) = x$ 在 $(0, 1)$ 内有唯一实根

**(2) 设函数 $f(x)$ 在 $[a, b]$ 上连续，在 $(a, b)$ 内可导，且 $f(a) = f(b) = 0$。证明：对任意实数 $k$，存在 $\xi \in (a, b)$，使得 $f'(\xi) = k f(\xi)$。**

证：令 $g(x) = e^{-kx} f(x)$
则 $g(a) = g(b) = 0$
由罗尔定理，存在 $\xi \in (a, b)$，使得 $g'(\xi) = 0$
$g'(x) = -k e^{-kx} f(x) + e^{-kx} f'(x) = e^{-kx}[f'(x) - k f(x)]$
所以 $g'(\xi) = e^{-k\xi}[f'(\xi) - k f(\xi)] = 0$
由于 $e^{-k\xi} \neq 0$，所以 $f'(\xi) - k f(\xi) = 0$，即 $f'(\xi) = k f(\xi)$

**(3) 设函数 $f(x)$ 具有二阶导数，且 $f(0) = 0$，$f''(x) > 0$。证明：对任意 $x_1, x_2 > 0$，有 $f(x_1 + x_2) > f(x_1) + f(x_2)$。**

证：不妨设 $x_1 \leq x_2$
令 $g(x) = f(x + x_2) - f(x) - f(x_2)$
则 $g(0) = f(x_2) - f(0) - f(x_2) = 0$
$g'(x) = f'(x + x_2) - f'(x)$
由中值定理，存在 $\xi \in (x, x + x_2)$，使得 $f'(x + x_2) - f'(x) = f''(\xi) x_2 > 0$
所以 $g'(x) > 0$，即 $g(x)$ 单调递增
故对任意 $x_1 > 0$，有 $g(x_1) > g(0) = 0$，即 $f(x_1 + x_2) > f(x_1) + f(x_2)$
