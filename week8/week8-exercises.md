# 第8周：练习题目

## 第一类换元法（凑微分法）

### 选择题
1. $\int \frac{1}{3x + 1} dx = $
   A. $\ln|3x + 1| + C$
   B. $\frac{1}{3} \ln|3x + 1| + C$
   C. $3 \ln|3x + 1| + C$
   D. $\frac{1}{3x + 1} + C$

2. $\int x e^{x^2} dx = $
   A. $e^{x^2} + C$
   B. $\frac{1}{2} e^{x^2} + C$
   C. $2 e^{x^2} + C$
   D. $x^2 e^{x^2} + C$

3. $\int \sin 2x dx = $
   A. $-\cos 2x + C$
   B. $\frac{1}{2} \cos 2x + C$
   C. $-\frac{1}{2} \cos 2x + C$
   D. $2 \cos 2x + C$

### 填空题
1. $\int \frac{x}{1 + x^2} dx = $ __________

2. $\int e^{-x} dx = $ __________

3. $\int \frac{\sin \sqrt{x}}{\sqrt{x}} dx = $ __________

### 解答题
1. 求$\int (2x + 1)^5 dx$。

2. 求$\int \frac{1}{x \ln x} dx$。

3. 求$\int \sin^3 x \cos x dx$。

4. 求$\int \frac{e^{\sqrt{x}}}{\sqrt{x}} dx$。

5. 求$\int \frac{x}{\sqrt{1 - x^2}} dx$。

6. 求$\int \frac{\arctan x}{1 + x^2} dx$。

7. 求$\int \sec^2 x \tan x dx$。

8. 求$\int \frac{1}{x^2} e^{1/x} dx$。

   A. $x = a \tan t$
   B. $x = a \sin t$
   C. $x = a \sec t$
   D. $x = a \cot t$

### 填空题

3. 求$\int \frac{1}{\sqrt{x^2 - 9}} dx$。

4. 求$\int \frac{x^2}{\sqrt{1 - x^2}} dx$。

5. 求$\int \frac{1}{x^2 \sqrt{1 + x^2}} dx$。

6. 求$\int \frac{1}{\sqrt{1 + e^{2x}}} dx$。

## 定积分的换元法

### 选择题
1. $\int_0^2 x \sqrt{4 - x^2} dx = $
   A. $\frac{8}{3}$
   B. $\frac{4}{3}$
   C. $\frac{16}{3}$
   D. $\frac{2}{3}$

2. $\int_0^{\pi/2} \sin^3 x \cos x dx = $
   A. $\frac{1}{4}$
   B. $\frac{1}{2}$
   C. $\frac{3}{4}$
   D. 1

3. $\int_1^e \frac{\ln x}{x} dx = $
   A. $\frac{1}{2}$
   B. 1
   C. $\frac{3}{2}$
   D. 2

### 填空题
1. $\int_0^1 \frac{x}{1 + x^2} dx = $ __________

2. $\int_0^{\pi/2} \cos^3 x \sin x dx = $ __________

3. $\int_0^2 \frac{1}{\sqrt{4 + x^2}} dx = $ __________

### 解答题
1. 计算$\int_0^1 (2x + 1)^3 dx$。

2. 计算$\int_0^{\pi/2} \sin 2x dx$。

3. 计算$\int_1^2 \frac{1}{x(\ln x)^2} dx$。

4. 计算$\int_0^a \sqrt{a^2 - x^2} dx$（$a > 0$）。

5. 计算$\int_0^1 \frac{x}{\sqrt{1 + x^2}} dx$。

6. 计算$\int_1^{\sqrt{3}} \frac{1}{x^2 \sqrt{1 + x^2}} dx$。

7. 计算$\int_2^4 \frac{\sqrt{x^2 - 4}}{x} dx$。

8. 计算$\int_0^{\pi/4} \tan^3 x \sec^2 x dx$。

## 综合应用题

1. 求$\int \frac{1}{\sqrt{e^x + 1}} dx$。

2. 求$\int \frac{x^3}{\sqrt{1 + x^2}} dx$。

3. 求$\int \frac{1}{x \sqrt{x^2 - a^2}} dx$（$a > 0$）。

4. 求$\int \frac{\sqrt{a^2 - x^2}}{x^2} dx$（$a > 0$）。

5. 求$\int \frac{x}{\sqrt{x^2 + 2x + 2}} dx$。

6. 计算$\int_0^{\pi/2} \cos^5 x dx$。

7. 计算$\int_{-a}^a \sqrt{a^2 - x^2} dx$（$a > 0$）。

8. 证明：$\int_0^{\pi/2} \sin^n x dx = \int_0^{\pi/2} \cos^n x dx$。

9. 求$\int \frac{1}{1 + e^x} dx$。

10. 求$\int \frac{x^2}{(x^2 + a^2)^{3/2}} dx$（$a > 0$）。
